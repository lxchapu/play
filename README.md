# PLAY - 自嘲型游戏人格测试

> P·L·A·Y Type Indicator · v1.0 — 一款基于矩阵评分的赛博共鸣诊断系统，通过极致的自嘲与毒舌文案，引发玩家对游戏意义的深度自省。

**在线访问**：[play.lxchapu.com](https://play.lxchapu.com/)

## 诊断流程

测试分为三个阶段，模拟一次沉浸式的"赛博体检"：

| 阶段    | 名称       | 说明                                                                      |
| ------- | ---------- | ------------------------------------------------------------------------- |
| Phase 0 | 传感器校准 | 前置题，确定玩家的主要生态位（单机/二游/竞技/杂食），决定后续专属题库加载 |
| Phase 1 | 病灶扫描   | 混合加载通用题 + 生态位专属题，打乱顺序，部分选项可触发动态分支追问       |
| Phase 2 | 终极诊断   | 输出包含人格代码、绰号、雷达图、灵魂终结评价及赛博处方                    |

## 四大诊断维度

PLAY 通过四个维度对玩家行为进行交叉分析，大写字母代表主导极性：

| 维度  | 大写极性    | 小写极性  | 核心问题                                     |
| ----- | ----------- | --------- | -------------------------------------------- |
| P / p | 执行力·死磕 | 随缘·逃避 | 你是爆肝卷王，还是赛博躺平大师？             |
| L / l | 逻辑·规划   | 直觉·玄学 | 你用脑子做最优解，还是凭感觉和运气行事？     |
| A / a | 攻击性·暴躁 | 温和·佛系 | 你是团战发动机/压力怪，还是气氛调解器？      |
| Y / y | 效率·功利   | 沉浸·情怀 | 你追求数值最优解和结果，还是享受过程和体验？ |

四字母组合 → 2⁴ = **16 种标准人格** + **4 种隐藏人格**。

## 生态位分流

Phase 0 的前置题将玩家分配至四个生态位，决定后续专属题库的加载：

| 生态位         | 题目侧重                             | 关键词               |
| -------------- | ------------------------------------ | -------------------- |
| Solo (单机)    | 挑战难度、世界观细节、成就系统       | 苦难、景观、深度     |
| Gacha (二游)   | 抽卡心态、角色情感连接、日常任务     | 厨力、非欧、保底     |
| PVP (竞技)     | 胜负执念、队友协作/冲突、版本强度    | 压力、上分、排位     |
| Variety (杂食) | 买爆心态、尝试新事物的频率、独立沙盒 | 喜加一、包工头、广度 |

## 评分算法

### 归一化得分

$$Score_D = \frac{\sum Weight_{D}}{\text{Max Weight}_{D}} \times 100$$

### 共鸣深度分级

| 区间   | 等级            | 描述                                               |
| ------ | --------------- | -------------------------------------------------- |
| 50-70  | 轻度映射 (Mild) | 游戏是你生活的调味剂，你玩得游刃有余。             |
| 71-90  | 深度共鸣 (Deep) | 游戏已成为你的第二现实，你在这里投射了真实的自我。 |
| 91-100 | 灵魂链接 (Soul) | 这里的代码比现实的砖块更懂你。                     |

### 动态分支

部分选项带有隐藏触发概率（如 `hackProbability`、`hamsProbability`、`mamaProbability`、`archProbability`），累计达到阈值时可激活对应的隐藏人格。同时部分选项可触发子题追问，形成"算法正在审视你"的追问感。
