
本文件为底层逻辑框架。所有题库设计、文案撰写、人格画像构建均须严格遵循此定义。

## 1. P - Persistence (执念 / 投入度)

**核心定义：玩家在虚拟世界中消耗生命的方式与目的。**

- **大写方向 \[P\] (Power-grinder / 主动产出)** 
    
    - **定义**：将游戏视为一种“主动劳动”。追求数值增长、全成就达成、进度的绝对领先。
    - **行为特征**：爆肝、日常全勤、红点消除强迫症。
    - **设计锚点**：**【确定性代偿】** 现实努力往往无果，玩家在游戏里寻找“投入必得回报”的底层安全感。
        
- **小写方向 \[p\] (Passive-slacker / 被动反馈)**
    
    - **定义**：将游戏视为“无责避难所”。拒绝进度压力，情绪驱动，随缘游玩。
    - **行为特征**：三分钟热度、赛博街溜子、遇到挑战光速下线。
    - **设计锚点**：**【责任感逃避】** 现实中的 KPI 已耗尽其意志。在游戏里任何“像上班”的任务都会触发其生理性厌恶。

## 2. L - Logic (逻辑 / 认知方式)

**核心定义：玩家面对未知与挑战时的底层防御机制。**

- **大写方向 \[L\] (Link-analyzer / 机制解构)**
    
    - **定义**：玩的是“系统”。信奉数据分析、逻辑推演，通过拆解机制寻求最优解。
    - **行为特征**：攻略党、数值计算、考据狂、追求版本 T0 解。
    - **设计锚点**：**【失控恐惧症】** 现实过于混沌且无理。玩家试图在透明的代码规则里，寻找“只要我够懂，就能赢”的虚假掌控感。
        
- **小写方向 \[l\] (Luck-believer / 感性驱动)**
    
    - **定义**：玩的是“反馈”。信奉直觉、审美与玄学。
    - **行为特征**：玄学赌狗、颜值至上、拒绝读说明书、凭肌肉记忆操作。
    - **设计锚点**：**【平庸宿命逆反】** 在被算法算计的现实里感到窒息，渴望成为“天选之子”，证明自己是不被逻辑限制的例外。

## 3. A - Aggression (侵略 / 社交位阶)

**核心定义：玩家在虚拟社交空间中的能量流向与权力确认方式。**

- **大写方向 \[A\] (Aggressive-pro / 竞技扩张)**
    
    - **定义**：向外的能量释放。追求支配权、胜负欲以及剥离他人体验的快感。
    - **行为特征**：分奴、族谱爆破手、压力怪、鱼塘屠夫。
    - **设计锚点**：**【现实话语权补偿】** 在现实位阶中处于被审判的劣势。通过在虚拟世界施加压力，对冲现实中“唯唯诺诺”的卑微。
        
- **小写方向 \[a\] (Autonomic-peaceful / 非对抗共存)**
    
    - **定义**：向内的能量吸收。追求协作、独立空间或绝对的非对抗环境。
    - **行为特征**：自闭种田、避世党、辅助奶妈、社交恐惧。
    - **设计锚点**：**【社交能量枯竭】** 现实中虚伪的客套和人际磨损使其长满厚茧。选择避世是因为太累了，只想守住一点点不被评价的尊严。

## 4. Y - Yearning (渴望 / 体验重心)

**核心定义：玩家消费内容时的终极渴望。**

- **大写方向 \[Y\] (Yield-seeker / 结果收益)**
    
    - **定义**：追求“产出”。效率至上，将资产（数值/成就）的快速获取视为唯一目的。
    - **行为特征**：剧情跳过、买号/代练代肝、跟风党、快餐化。
    - **设计锚点**：**【过程丧失异化】** 被现实的“唯结果论”深度异化，彻底丧失了“感受过程”的能力。潜意识认为没有即时收益的行为就是在浪费生命，玩游戏变成了赛博投资。
        
- **小写方向 \[y\] (Yarn-spinner / 虚构沉浸)**
    
    - **定义**：追求“体验”。追求与世界观、NPC 或故事背后的情感流动。
    - **行为特征**：剧情落泪、风景党、设定原教旨主义、电子暖暖。
    - **设计锚点**：**【现实情感难民】** 现实生活中缺乏深度的情感羁绊和共鸣，导致他们长出了极其敏感的虚拟触角。将无处安放的温情倾注在像素上，在赛博废墟里寻找灵魂共振。
